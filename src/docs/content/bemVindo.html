Agradecemos pelo interesse no Software do Teleprompter N@Moral Produções Audio Visuais. Estamos continuamente otimizando, os itens que compõe um sistema de Teleprompter ( Software, Hardware e Mecânica ). Este software foi desenvolvido utilizando-se a linguagem Java , com o objetivo de poder ser utilizado com diversos sistemas operacionais desde que esteja instalado na maquina a JVM ( Java Virtual Machine ) ou seja o JRE que pode ser adquirido sem custo do site da Sun.
